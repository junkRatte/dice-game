Rules ↓

The game has 2 players, playing in rounds. In each turn, a player rolls a dice as many times as they whish. Each result gets added to their CURRENT score and after the player presses HOLD, the score from CURRENT gets added to the GLOBAL score. BUT if a player throws a 1 on a dice then their CURRENT score is lost and it's the next players turn. The game ends when a player reaches the score that is set in the FINAL SCORE window (if no value is set, the default winning score is 100).
